# Required Dashboard UI Assets

Add these PNG files directly inside the `image` folder. Use transparent-background PNGs where possible.

## Brand And Top Bar

| Filename | What To Add |
| --- | --- |
| `logo_graduation_cap.png` | Blue graduation cap logo, like the sidebar header. |
| `admin_avatar.png` | Circular admin/student avatar illustration. |
| `top_calendar.png` | Small calendar outline icon. |
| `top_bell.png` | Bell notification icon with a small red badge. |
| `status_shield.png` | Shield/check icon for the system status card. |

## Sidebar Icons

| Filename | What To Add |
| --- | --- |
| `nav_dashboard.png` | White home/dashboard icon. |
| `nav_register_student.png` | White single-user-plus icon. |
| `nav_manage_students.png` | White group/users icon. |
| `nav_view_attendance.png` | White calendar/check icon. |
| `nav_exit.png` | Red/white power icon. |

## Metric Cards

| `sparkline_present.png` | Small green line chart sparkline. |

## Action Cards

| Filename | What To Add |
| --- | --- |
| `card_register_student.png` | Green user-plus icon. |
| `card_face_id.png` | Blue face scan icon. |
| `card_fingerprint.png` | Cyan/blue fingerprint icon. |
| `card_rfid.png` | Purple RFID card icon. |
| `card_view_reports.png` | Blue analytics/bar chart icon. |
| `illustration_reports.png` | Report document with chart illustration. |
| `card_exit_power.png` | Red power button icon. |
| `illustration_exit_door.png` | Light gray door/exit illustration with red arrow. |

Recommended sizes:

- Small top/sidebar icons: 64x64 PNG.
- Card and metric icons: 96x96 PNG.
- Report/exit illustrations: 180x140 PNG or larger.
- Sparklines: about 120x56 PNG.
